# Personal Portfolio

A simple responsive portfolio website built with HTML and CSS.

## Project Structure

- `index.html` — main landing page with intro, projects, work, education, and feedback sections.
- `projects.html` — dedicated projects list page with descriptions.
- `articles.html` — articles page modeled on experience data.
- `contact.html` — contact form and contact details page.
- `style.css` — shared CSS styles for layout, grid cards, and responsive design.

## Features

- responsive grid layout with `auto-fit` and `minmax` for content cards
- clean header navigation across pages
- styled form controls on contact page
- accessible semantics with sections, articles, and lists

## Local Setup

1. Clone or copy the folder locally.
2. Open `index.html` in a web browser.
3. Navigate to `projects.html`, `articles.html`, and `contact.html` via the top navigation.

## Customize

- Update content in HTML files.
- Modify colors, spacing, and responsive behavior in `style.css`.
- Add images and additional sections as needed.

## Notes

- This project is lightweight and frontend-only; no server required.
- Ensure all page links are relative and in the same folder.
