# FrontEnd Projects

This repository contains simple frontend website projects for learning and portfolio use.

## Folder structure

- `Basic HTML Website/` — simple multi-page site (index, projects, articles, contact).
- `Personal Portfolio/` — improved portfolio version with consistent styling and responsive card layout.

## Basic HTML Website

Files:
- `index.html` — basic home page with mods for sections and lists.
- `projects.html` — projects page (generated with descriptions).
- `articles.html` — articles page (generated from pinned work items).
- `contact.html` — simple contact form page.

## Personal Portfolio

Files:
- `index.html` — main portfolio page with navigation, intro, content grid, and reviews.
- `projects.html` — projects listing page.
- `articles.html` — articles page.
- `contact.html` — contact form page.
- `style.css` — shared styling, responsive grid, cards, and contact page styles.
- `README.md` — project docs.

## Run locally

1. Open any HTML file in a browser.
2. Use the nav links to move between pages.

## Tips

- Edit your name, contact, and text in HTML files.
- Customize visual design in `style.css`.
- Use a live server extension for faster dev feedback (vscode Live Server, etc.).
